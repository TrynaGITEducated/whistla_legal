📢 See something? Say something.

Whistla is a community-driven platform for reporting car issues — lights left on, alarms sounding, bad parking, and more.

🚀 How It Works

1. See an issue in a parking lot
2. Report it with location and plate number
3. Alert the community to help find the owner
4. Mark as resolved when fixed
